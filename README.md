# Bhagath Pranav — Portfolio

A dark, minimalist portfolio inspired by the **Sawad** Framer template — built with Astro, TypeScript, and zero clutter.

[🔗 Live Site](bhagathpranav.vercel.app) · [📂 All Projects](https://github.com/BhagathPranav?tab=repositories)

---

## ✨ What This Is

A single-page portfolio site that showcases my work as a **Data Analyst & IT Developer**. No contact forms, no popups, no noise — just clean typography, smooth scroll animations, and projects that speak for themselves.

### What's Inside

| Section | Content |
|---------|---------|
| **Hero** | Massive stacked headline + tagline |
| **Stats** | Animated counters (Years, Projects, Tech Stacks) |
| **Skills** | Uppercase comma-separated tech tags |
| **Featured Projects** | 3 highlight cards: AI-chart, Pinventory, Vibe Portfolio |
| **Experience** | Present role with bullet-point responsibilities |
| **Footer** | Name, tagline, minimal nav |

---

## 🚀 Tech Stack

- **Framework:** [Astro](https://astro.build) — fast, static, zero JS by default
- **Language:** TypeScript
- **Styling:** Custom CSS (dark theme, no frameworks)
- **Animations:** Intersection Observer + CSS transitions
- **Deployment:** Vercel
