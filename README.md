🎨 Design Notes
Dark theme: #111 background, pure white headings, muted gray body text
Typography: Bold oversized headlines (Sawad-style), tight letter-spacing
Spacing: Generous but intentional — 120px+ between major sections
Animations: Scroll-triggered fade-ins with staggered card reveals
Zero contact UI: No forms, no buttons, no distractions
